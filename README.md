# uk2en — автопереклад надиктованого тексту з української на англійську

Маленька фонова утиліта для macOS. Працює в парі з [Handy](https://handy.computer/)
(або будь-яким іншим інструментом, що пише в буфер обміну).

## Як це працює

1. Ви натискаєте хоткей Handy і диктуєте українською
2. Handy транскрибує і пише текст у буфер обміну, потім робить Cmd+V
3. uk2en daemon помічає кирилицю, перекладає офлайн через Argos Translate
4. Замінює буфер на англійський і видаляє оригінал клавіатурним скороченням
5. У полі лишається тільки англійський текст

Усе локально, безкоштовно, без інтернету (після першого запуску).

## Встановлення

```bash
./install.sh
```

Інсталятор:
- створює venv в `~/.local/uk2en/`
- ставить `argostranslate` і `pyobjc`
- завантажує модель uk→en (~80 МБ, разово)
- ставить LaunchAgent для автозапуску при логіні (режим `--terminal`)

## Дозволи

Перший раз коли daemon спробує натиснути Cmd+V, macOS запитає дозвіл
**Accessibility** для Python. Видайте його:

System Settings → Privacy & Security → Accessibility → увімкнути для Python
(той що в `~/.local/uk2en/venv/bin/python3`).

## Режими запуску

| Прапор | Коли використовувати |
|---|---|
| `--terminal` | WebStorm / iTerm / будь-який термінал. Ctrl+U очищає рядок, Cmd+V вставляє переклад. **За замовчуванням у LaunchAgent.** |
| *(без прапора)* | Звичайні текстові поля (Notion, браузер тощо). Cmd+Z скасовує українську вставку, Cmd+V вставляє переклад. |
| `--paste-only` | Якщо Handy налаштовано на "тільки копіювати" — daemon тільки робить Cmd+V без скасування. |
| `--copy-only` | Тільки оновити буфер, нічого не натискати. |

## Тестування

```bash
# Скопіюйте будь-який український текст у буфер, потім:
~/.local/uk2en/uk2en_daemon.py --once -v --terminal
```

## Керування

```bash
# Логи в реальному часі
tail -f ~/.local/uk2en/daemon.log

# Перезапуск
launchctl kickstart -k gui/$UID/com.user.uk2en

# Зупинити
launchctl unload ~/Library/LaunchAgents/com.user.uk2en.plist

# Запустити знову
launchctl load ~/Library/LaunchAgents/com.user.uk2en.plist

# Фікс SSL-сертифікатів (якщо модель не завантажується)
./install.sh --fix-certs
```

## Параметри daemon

```
--terminal      Режим для терміналу: Ctrl+U + Cmd+V (за замовч. у LaunchAgent)
--paste-only    Cmd+V без скасування (для Handy в copy-only режимі)
--copy-only     Тільки оновити буфер, нічого не натискати
--once          Перекласти буфер один раз і вийти (тест)
--poll 0.12     Інтервал опитування буферу, секунди
-v              Verbose-логи
```

## Видалення

```bash
./uninstall.sh
```

## Відомі обмеження

- **Якість перекладу**: Argos Translate uk→en — це OPUS-MT модель. Якість
  робоча, але іноді спотикається на ідіомах і складних реченнях.
  Для кращої якості можна замінити на DeepL Free API (500 000 символів/місяць
  безкоштовно) — потрібно переписати функцію `translate()`.

- **Затримка**: 100–300 мс між українським текстом і англійським (залежить
  від довжини тексту і швидкості Argos).

## Архітектура

- `uk2en_daemon.py` — Python скрипт, що використовує `AppKit.NSPasteboard`
  для опитування буферу обміну і AppleScript (`osascript`) для імітації клавіш
- Детектор мови: Ukrainian-specific символи `іїєґ` — не зустрічаються в
  російській/болгарській/сербській, тому false-positives виключені
- LaunchAgent (`~/Library/LaunchAgents/com.user.uk2en.plist`) запускає
  daemon при логіні і перезапускає якщо впаде
- Argos Translate (`~/argos-translate/`) — офлайн моделі перекладу
